# Quest Roulette

モンスター周回を管理するランダム選出ツールです。

## 特徴

- **直感的なフロー**: FUTURE（次回）→ CURRENT（今回）→ PREVIOUS（前回）の順に並び、タスクの流れがひと目で分かります。
- **自動計算**: モンスターを決定するたびに、設定画面の残り回数が自動で減ります。
- **簡単追加**: 「＋」ボタンで対象モンスターを増やすことができます。

## 導入方法

1. `index.html` を任意のフォルダに保存します。
2. ブラウザで開くだけで動作します。

## 画像の表示について

モンスターの画像を表示したい場合は、HTMLファイルと同じフォルダに画像（例：`redau.png`）を置き、JavaScript内の `imageMap` 部分を書き換えてください。
